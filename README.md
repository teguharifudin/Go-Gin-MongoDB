# Go Gin MongoDB

a CRUD API with Go's Gin library using MongoDB.

## Installation

```
git clone https://github.com/teguharifudin/Go-Gin-MongoDB.git Go-Gin-MongoDB
cd Go-Gin-MongoDB
```

### Testing

```
go run main.go 
```
